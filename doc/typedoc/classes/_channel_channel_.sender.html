Class Sender<T>

Sends values into a channel. Allows senders to optionally await for the reciever to receive each value. Senders are Marshal.

// Synchronized
await sender.send(1)
await sender.send(2)
await sender.send(3)
await sender.end()

// Async-Buffered
sender.send(1)
sender.send(2)
sender.send(3)
await sender.end()
